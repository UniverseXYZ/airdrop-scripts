# Airdrop Scripts

Scripts to retrieve airdrop users and polymorph owners.

For polymorph uses the graph `https://thegraph.com/explorer/subgraph/schmackofant/polymorphs-nft`

## How to use

For creating an airdrop file run the following command:

` yarn run airdrop`

For polymorph owners file run the following command:

` yarn run polyowners`
